## **PENGANTAR**

Aplikasi StarWars App di kembangkan oleh [Imam Abdul Azis](https://github.com/imamabdulazis) untuk menyelesaikan **Senior Frontend Test** dari [Worlder Group](https://worldergroup.com)

## **Mulai**

### **StarWarsAPI**

Ambil dan tampilkan informasi kapal luar angkasa dan profil orang. Api ini dapat mengambil data StarWars melalui [API URL](https://swapi.dev)

</br>
</br>

### **Unduh Projek**

```shell
git clone https://github.com/imamabdulazis/StarWarsApp.git
```

</br>
</br>

### **Install Dependencies**

```shell
npm install atau yarn install
```

</br>
</br>

### **Run**

```shell
yarn iphone8 atau yarn android
```

</br>
</br>

### **Requirement**

Persyaratan meliputi:

- Android 5.1 (Lollipop) atau lebih
- Ukuran Layar of 7 inches atau kurang
- React native (0.64 atau lebih)
  </br>
  </br>

### **Features**

Aplikasi ini memiliki beberapa fitur dan komponen yang dapat digunakan oleh pengguna

- Daftar
- Gabung
- Membuat grup
- Tambahkan anggota grup
- Undang anggota grup
- Tampilkan pemberitahuan dalam aplikasi dari undangan grup
- Izinkan pengguna yang diundang untuk bergabung dengan grup
- Ambil dan tampilkan lokasi GPS terakhir yang diketahui
- Ambil dan tampilkan informasi kapal luar angkasa
  </br>
  </br>
